# Python购物商城系统（源码+SQL文件）

## 核心功能
- 商品浏览
- 商品详情
- 个人信息管理
- 购物车管理
- 订单管理
- 商品类型管理
- 订单支付

## 适用场景
- 电商平台
- 网上商店
- 移动购物应用

## 技术特点
- 使用 Django 作为后端框架，确保高效的数据处理和用户交互
- 采用 MySQL 数据库，保证数据的稳定性和安全性
- 前端采用 Vue 框架，提供良好的用户界面和交互体验
